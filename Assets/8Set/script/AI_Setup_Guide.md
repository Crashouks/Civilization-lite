# Штучний Інтелект для CIVlite - Інструкція з налаштування

## Що створено

Я створив повноцінну систему штучного інтелекту для вашої гри, яка включає:

### 1. **CivilizationAI.cs** - Головний AI менеджер
- Керує цілою цивілізацією
- Приймає рішення про війну та мир
- Створює юнітів та міста
- Налаштовує стратегію поведінки

### 2. **DiplomacySystem.cs** - Дипломатична система
- Відстежує відносини між цивілізаціями
- Автоматично оголошує війну
- Керує дипломатичним статусом
- Впливає на поведінку AI

### 3. **CombatSystem.cs** - Бойова система
- Реалізує атаку юнітів
- Розраховує пошкодження
- Керує бойовими механіками
- Інтегрується з здоров'ям юнітів

### 4. **UnitHealth.cs** - Система здоров'я
- Додає здоров'я юнітам
- Показує health bar
- Відстежує пошкодження

### 5. **Оновлений UnitAI.cs** - Розумний AI для юнітів
- Аналізує ситуацію на полі бою
- Приймає рішення про атаку/рух/розвідку
- Працює з різними стратегіями

## Як налаштувати

### Крок 1: Додайте компоненти до сцени

1. **Створіть пустий об'єкт** з назвою "AI Manager"
2. **Додайте компоненти:**
   - `CivilizationAI`
   - `DiplomacySystem` 
   - `CombatSystem`

### Крок 2: Налаштуйте CivilizationAI

У інспекторі CivilizationAI вкажіть:

**Налаштування цивілізації:**
- `Civilization Name` - назва AI цивілізації
- `Civilization Color` - колір юнітів та міст
- `Aggression Level` - рівень агресії (0 = мирний, 1 = агресивний)
- `Expansion Priority` - пріоритет розширення (0 = низький, 1 = високий)
- `War Declaration Chance` - шанс оголосити війну за хід

**Префаби юнітів:**
- `Settler Prefab` - префаб поселенця
- `Warrior Prefab` - префаб воїна
- `Archer Prefab` - префаб лучника (опціонально)

### Крок 3: Налаштуйте CombatSystem

**Налаштування бою:**
- `Combat Animation Duration` - тривалість анімації атаки
- `Combat Effect Prefab` - ефект для візуалізації атаки (опціонально)

### Крок 4: Налаштуйте UnitHealth для юнітів

Для кожного префаба юніта додайте компонент `UnitHealth`:
- `Max Health` - максимальне здоров'я
- `Health Bar Prefab` - префаб для відображення здоров'я (опціонально)
- `Show Health Bar` - показувати health bar

### Крок 5: Перевірте інтеграцію з Program1

Переконайтеся, що у вашому `Program1.cs` є:
- Поле `isAtWar` для відстеження стану війни
- Методи `RemoveUnit()` та `RegisterCity()`
- Доступ до `allUnits` та `allCities`

## Як це працює

### Автоматичний AI цикл
1. **Аналіз дипломатії** - AI перевіряє, чи оголошувати війну
2. **Створення юнітів** - у містах створюються нові юніти
3. **Керування юнітами** - кожен юніт приймає розумні рішення
4. **Розширення** - поселенці засновують нові міста

### Поведінка юнітів
- **Воїни:** шукають ворогів, атакують, патрулюють
- **Поселенці:** шукають місця для нових міст
- **Лучники:** атакують з дальньої дистанції

### Дипломатія
- **Відносини:** від -100 (війна) до +100 (союз)
- **Статуси:** War, Hostile, Neutral, Friendly, Allied
- **Події:** битви, захоплення міст впливають на відносини

## Можливості налаштування

### Зміна агресивності AI
```csharp
// У CivilizationAI
aggressionLevel = 0.8f; // Дуже агресивний
warDeclarationChance = 0.5f; // 50% шанс війни за хід
```

### Додавання нових типів юнітів
```csharp
// У CombatSystem.InitializeUnitStats()
unitStats["Swordsman"] = new UnitStats("Swordsman", 25, 12, 10, 2, 1);
```

### Налаштування дипломатії
```csharp
// У DiplomacySystem
peaceDuration = 15f; // Триваліший мир
warDeclarationThreshold = 0.7f; // Вищий поріг для війни
```

## Поради для тестування

1. **Почніть з низької агресії** (0.2-0.3) щоб побачити розширення AI
2. **Спостерігайте за логами** - AI повідомляє про всі рішення
3. **Тестуйте війну** - збільште агресію до 0.8+
4. **Перевірте дипломатію** - слідкуйте за змінами відносин

## Поширені проблеми

### AI не створює юнітів
- Перевірте, чи призначені префаби в CivilizationAI
- Переконайтеся, що у AI є міста

### Юніти не атакують
- Перевірте, чи додано CombatSystem до сцени
- Переконайтеся, що юніти мають компонент UnitHealth

### Дипломатія не працює
- Перевірте, чи додано DiplomacySystem
- Переконайтеся, що Program1 має поле isAtWar

3. **Додайте торгову систему** для впливу на дипломатію
4. **Реалізуйте перемир'я** та мирні договори
